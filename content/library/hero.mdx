# Hero

Hero section, crafted with React and Tailwind CSS. The hero section is a full-width container with a background image and a title.

## Preview

<ComponentPreview name="hero" />

## Configuration

Add the animation to your **_tailwind.config.ts_** file.

<CodeBlockWrapper size="full">
```ts {4-7} {10}
  theme: {
    extend: {
      keyframes: {
        "hero-in": {
          from: { opacity: "0", transform: "translateY(1.5rem)" },
          to: { opacity: "1", transform: "translateY(0)" },
        },
      },
      animation: {
        "hero-in": "fade-in 0.6s ease forwards",
      },
    },
  }
```
</CodeBlockWrapper>

**_Note:_** This is not the entire configuration file, only the relevant parts are shown. Make sure to include the new configuration inside the **_theme_** object.
